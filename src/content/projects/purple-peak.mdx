## About

Purple Peak is a collection of finance calculators.

Currently have a mortgage calculator, home affordability calculator, and an investment calculator.

## Motivation

I started getting restless since finishing my previous project and wanted an excuse to try out Shadcn/UI.

Finance calculators seemed like a great way to experiment with cool UI components without having to build a backend + they're already something I waste too much time playing around with.

## Tech Stack

- Next.js
- Tailwind
- Shadcn/UI
- Cloudflare Workers
- Cloudflare KV
